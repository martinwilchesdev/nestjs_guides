## Instalar NestJS

```bash
$ npm install @nest/cli
```

## Inicializar un nuevo proyecto

```bash
$ nest new <project_name>
```

## Ejecutar la aplicacion

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Comandos

#### Crear un nuevo modulo

```bash
$ npm g mo <module_name>
```
